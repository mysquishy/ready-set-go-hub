## Typography

### Headings
- **Font:** System font stack (optimized per platform)
- **Weight:** 700 (Bold) for h1-h3, 600 (Semibold) for h4-h6
- **Style:** Clean, modern, sans-serif

### Body Text
- **Font:** System font stack
- **Weight:** 400 (Regular)
- **Line Height:** 1.6

### Code
- **Font:** Monospace (SF Mono, Consolas, Monaco)
- **Background:** Light gray (#F5F5F5)

## Design Principles

1. **Clean & Modern:** Minimalist design with focus on content
2. **Purple Gradient Accents:** Signature gradient used sparingly
3. **Mobile-First:** Responsive design for all devices
4. **Smooth Animations:** Subtle, performant animations (Framer Motion)
5. **High Contrast:** Accessibility-first design (WCAG AA)

## Voice & Tone

- **Professional but friendly:** Technical authority without jargon
- **Action-oriented:** Focus on results and speed
- **Developer-first:** Speak the language of developers
- **Confident:** Show expertise without arrogance

## Website Guidelines

### Hero Section
- Large, bold headline
- Purple gradient text on key words
- Clear CTA button (coral accent color)
- Product preview image or video

### Features Section
- Icon + title + description cards
- 3-column grid on desktop, stacked on mobile
- Purple accent on icons

### Pricing Section
- Clean pricing cards
- Highlight recommended tier
- Show bundle savings prominently

### Footer
- Links to products, docs, support
- Social media icons
- Copyright notice
